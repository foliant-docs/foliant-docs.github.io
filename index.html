Write in Markdown

Foliant uses Markdown, so writing new docs and migrating from MkDocs or Pandoc is easy.

Use the Power of Preprocessors

Preprocessors allow you to include documents, embed diagrams, generate DB schemes, and more. Choose from existing preprocessors write your own.

Ship to many targets at once

Foliant allows you to build the docs in html, pdf, and docx formats without editing the source for each target. Powered by MkDocs, Pandoc, and Slate.
